# Game Rules

## 已支持

- 标准初始布局
- 合法走法校验
- 吃子
- 将军检测
- 将死 / 和棋 / 逼和判定
- 升变
- 本地双人和联网模式共用同一套规则引擎

## 当前实现方式

- 前端使用 `chess.js` 作为规则引擎
- `ChessEngine` 负责把引擎状态序列化为 `GameState`
- 第一版后端不做完整规则裁定，只做房间回合、玩家身份和消息结构校验

## 已知限制

- 第一版后端并不是完全可信裁判，客户端仍然负责生成下一版 `GameState`
- `leaveRoom` 目前只有占位行为，尚未完成完整退房流程
- 观战模式尚未实现
